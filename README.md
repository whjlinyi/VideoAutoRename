# VideoAutoRename
> 用以视频下载后重命名:
>
> * 放到改名路径或者添加到变量
> * 脚本自动命名规则 : 按照剧集 集数命名 , 剧集数量必须小于100
> * 操作方式 : 按照`.后缀`来输入匹配 , 匹配完成后输入`Yes`重命名
